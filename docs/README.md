# README

> 我制作的docsify工程文件模板
>
> 结构清晰易用, 预先集成了一些好用的插件
> 
> 直接在该文件夹内进行修改构建你自己的文档即可
> 
> 不需要的插件或配置选项推荐在`index.html`中注释掉或删掉



## Docsify 基本指令

`docsify serve` - 运行此命令以在本地启动 Docsify 服务器，并在浏览器中查看文档网站。
